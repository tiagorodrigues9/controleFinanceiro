# Ícones PWA

Este diretório deve conter os ícones para o PWA em diferentes tamanhos:

- icon-72x72.png
- icon-96x96.png  
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

Você pode gerar esses ícones usando ferramentas como:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

Os ícones devem ter um design simples e reconhecível, com o tema azul (#1976d2) do aplicativo.
